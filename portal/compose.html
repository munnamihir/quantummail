QuantumMail MVP demo
Generate PQC Keys

Create a secure message link

Encrypt in the browser, store only ciphertext on the server, share a link in any email client.

Uses ML-KEM-768 to encapsulate an AES-256-GCM key to the recipient’s public key.

Derives an AES key via PBKDF2. Good for quick sharing with external recipients.

—

How it works

1) You encrypt locally in your browser.

2) Server stores only encrypted bytes (no keys).

3) Recipient opens the link and decrypts locally.

Security note: This repo is a demo MVP and has not been audited. Use it for demos/pilots, not for production sensitive data.
