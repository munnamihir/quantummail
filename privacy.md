# Privacy Policy

QuantumMail is a privacy-focused browser extension designed to perform
client-side encryption and decryption.

## Data Collection
QuantumMail does not collect, store, or transmit personal data, email
content, or user identifiers.

## Data Usage
All processing occurs locally within the user's browser and is initiated
explicitly by the user.

## Data Sharing
QuantumMail does not share data with third parties.

## Data Retention
No user data is retained. Any locally stored configuration data can be
cleared by removing the extension.

## Contact
For questions about this privacy policy, contact the project maintainer.
